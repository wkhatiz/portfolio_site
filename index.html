Projects

Walerdle

This is a wordle game created using Next/React.
The background is generated dynamically using the Unsplash API using the Next getServerSideProps function.
The game uses Next API routes to implement the logic for generating a random word, getting the corresponding colors for each guess, and returning the correct answer at the end of the game.
Session data such as the session id and corresponding correct answer, is stored in a Firebase Firestore DB.

HDIGOOTHS

Town Hall station is currently a mess of connecting buildings, diverging paths and multiple unmarked exits leading to busy streets. The goal of this web app is to map out all the possible exits out of Town Hall Station. I am using React Native to complete the first build which is just a POC mapping out only the station itself. Following releases will expand the scope to the adjoining buildings.
